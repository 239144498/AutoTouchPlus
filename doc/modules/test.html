test (description, tests, setup, teardown)
Calls test functions in a table (indexed by name) and prints failures

Parameters:

  • description general description of tests
  • tests table of test functions (function that makes assertions)
  • setup function that sets up the test environment
  • teardown function that tears down/cleans up the test environment

Usage:

    local failed = false
failed = failed or test('my test description', {
         myTest1 = function() assert(true, "my error message1") end,
         myTest2 = function() assert(true, "my error message2") end,
       })
